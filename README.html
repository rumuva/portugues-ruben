Cómo publicar en GitHub Pages

  1. Crea un repositorio en GitHub llamado portugues-ruben.
  2. Sube el contenido del ZIP a la rama main (sin carpetas adicionales).
  3. Ve a Settings → Pages y elige Deploy from a branch, rama main, carpeta /root. Guarda.
  4. Tu web quedará disponible en: https://TU_USUARIO.github.io/portugues-ruben/

Edita los contenidos en modules/* y añade audios MP3 en assets/audio.
